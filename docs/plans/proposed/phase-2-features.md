# Phase 2 Features

Quality-of-life features and Android platform integrations.

## Why

Once core polish is complete, these features make the app feel native and reduce friction for frequent use.

## Features

- [ ] Quick Settings tile — record from notification shade without opening the app
- [ ] Home screen widget — one-tap record button
- [ ] Result notification — show recognition result as a notification (pairs with tile/widget)
- [ ] Haptic feedback on recording start/stop
- [ ] Recordings screen — `GET /recordings`, playback via `GET /recordings/:id`, `DELETE /recordings`
- [ ] GitHub Actions CI — build + lint on push
- [ ] Dark theme matching MixMates branding
